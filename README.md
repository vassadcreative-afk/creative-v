# ✦ Creative V

Aplicativo web de gestão de clientes e tarefas, totalmente responsivo — funciona em desktop, tablet e mobile.

## 🚀 Como usar

Acesse direto pelo GitHub Pages ou abra o arquivo `index.html` no navegador. Nenhuma instalação necessária.

## ✨ Funcionalidades

- **Hoje** — visão geral do dia com tarefas e eventos
- **Quadro** — kanban com colunas: A fazer, Em andamento, Aguardando, Concluído
- **Calendário** — visualização mensal com marcadores de tarefas
- **Clientes** — sidebar com filtro por cliente
- **Google Calendar** — integração opcional via OAuth

## 📱 Responsivo

| Tela | Comportamento |
|------|--------------|
| ≥ 1400px | Layout expandido, sidebar larga |
| 1024–1399px | Desktop padrão |
| 768–1023px | Tablet landscape, kanban 2 colunas |
| 640–767px | Tablet portrait compacto |
| ≤ 639px | Mobile com bottom nav e FAB |
| ≤ 375px | Mobile pequeno otimizado |

## 🌐 Deploy com GitHub Pages

1. Faça o fork ou upload deste repositório
2. Vá em **Settings → Pages**
3. Em *Source*, selecione `main` e pasta `/ (root)`
4. Clique em **Save** — o link será gerado automaticamente

## 🛠 Tecnologias

- HTML5 + CSS3 + JavaScript puro
- [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) (Google Fonts)
- Sem dependências externas
